React 常用Hooks

发表于2022-10-26|更新于2022-10-27|React
|阅读量:

1. React.useState()

(1). State Hook让函数组件也可以有state状态, 并进行状态数据的读写操作
(2). 语法: const [xxx, setXxx] = React.useState(initValue)
(3). useState()说明:
参数: 第一次初始化指定的值在内部作缓存
返回值: 包含2个元素的数组, 第1个为内部当前状态值, 第2个为更新状态值的函数
(4). setXxx()2种写法:
setXxx(newValue): 参数为非函数值, 直接指定新的状态值, 内部用其覆盖原来的状态值
setXxx(value => newValue): 参数为函数, 接收原本的状态值, 返回新的状态值, 内部用其覆盖原来的状态值

2. React.useReducer(reducer, action)

(1). 对于复杂的state操作逻辑，嵌套的state的对象，推荐使用useReducer

3. React.useEffect()

(1). Effect Hook 可以让你在函数组件中执行副作用操作(用于模拟类组件中的生命周期钩子)，组件渲染完成后执行；

(2). React中的副作用操作:

​ 发ajax请求数据获取

​ 设置订阅 / 启动定时器

​ 手动更改真实DOM

(3). 语法和说明:

  • 没有依赖：初始化和组件刷新时都会执行；
  • 空依赖[]：只在初始化时执行；
  • 指定依赖[watchValue]：初始化和依赖watchValue发生改变时执行；
  • 返回的函数：返回的函数只在组件卸载前执行（componentWillUnmount）；

(4). 可以把 useEffect Hook 看做如下三个函数的组合:componentDidMount() componentDidUpdate() componentWillUnmount()

4. React.useMemo()

(1). 一般用于编写比较耗费资源且无需重复执行的代码，以达到优化性能目的，返回值为回调函数的结果(类似vue的computed)；

  • 空依赖[]：只在初始化时计算，组件刷新时都是从缓存中读取；
  • 指定依赖[watchValue]：初始化和依赖watchValue发生改变时计算，否则从缓存中读取；
  • 没有依赖：初始化和组件刷新时都会重新计算；

5. React.useCallback()

(1). useCallback返回传入的函数，不需要显性返回，一般用于第一事件处理函数；

  • 空依赖[]：返回一个闭包函数，刷新时永远返回缓存函数；
  • 指定依赖[watchValue]：初始化和依赖watchValue发生改变时返回新的函数，否则返回缓存函数；
  • 没有依赖：每次都返回一个新的函数；

6. React.useLayoutEffect()

(1). useLayoutEffect为useEffect的同步版本，会在页面渲染前执行，会阻塞页面渲染(类似componentWillMount)；

(2). 语法和说明:

  • 没有依赖：初始化和组件刷新时都会执行；
  • 空依赖[]：只在初始化时执行；
  • 指定依赖[watchValue]：初始化和依赖watchValue发生改变时执行；
  • 返回的函数：返回的函数只在组件卸载前执行（componentWillUnmount）；

7. React.useRef()

(1). Ref Hook可以在函数组件中创建ref对象，存储/查找组件内的标签或任意其它数据
(2). 语法: const refContainer = useRef()
(3). 作用:保存标签对象,功能与React.createRef()一样

8. React.useContext()

之前使用Consumer

(1). Context Hook可以在函数组件接收context

九. 自定义Hook

  • 自定义Hook是一个函数，名称一般以use开头，这样可以一眼看出符合Hook的规则；
  • 函数内部也可以使用其它Hook；
  • 自定义Hook解决了以前在React组件中无法灵活共享逻辑的问题。

十. 第三方Hook

  • react-router

    • useParams() 返回URL参数的键/值对的对象。使用它来访问当前路由Route的match.params

    • useHistory() 返回 history 对象

    • useLocation() 返回 history.location 对象

    • useRouteMatch() 返回 match 对象

  • react-redux
    • useDisPatch() 返回dispatch(action)方法
    • useSelector() const props = useSelector((state) => state)
    • useStore() 这个Hook直接获取到了Redux store的引用，其中，useStore().dispatch 其实等同于 useDispatch()
